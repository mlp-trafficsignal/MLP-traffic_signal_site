# 公開用データファイル

このフォルダは、公開時にサイトが読み込むデータ置き場です。

- `entries.json`: 紹介1件
- `makers.json`: メーカーと別名
- `categories.json`: 都道府県ごとのカテゴリ
- `top-images.json`: サイト全体トップ画像
- `pref-images.json`: 都道府県一覧画像、県ページピックアップ
- `terms.json`: 基礎用語
- `photo-tags.json`: 写真タグ候補

今の管理画面は試作用としてブラウザ内に保存します。公開用へ移すときは、管理画面のバックアップを書き出して、このフォルダの各JSONへ分けて配置する流れにします。
